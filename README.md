# Cricket Stadium Seat Booking System

## 📌 Description
This is a JavaFX-based application to manage and book cricket stadium seats.

## 🚀 Features
- Add new seats
- Book seats
- View all seats
- Search and filter seats

## 🛠 Technologies Used
- Java
- JavaFX
- Maven

## ▶️ How to Run
mvn clean javafx:run

## 📂 Project Structure
- MainApp.java → Entry point
- Booking.java → Booking logic
- Seat.java → Seat model
- StadiumService.java → Business logic
